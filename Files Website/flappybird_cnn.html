Beating Flappy Bird Using Convolutional Neural Networks


I used a Convolutional Neural Network with Reinforcement Learning to play Flappy Bird

I choose to create a convolutional deep q-neural network and train it with reinforcement learning to play the game as long as I can. I first created the game environment using pygame and coded in rewards and punishments. Then set up my convolutional neural network, and trained it to play Flappy Bird over many iterations.
